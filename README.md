# 🦉 Duolingo: Guía de Optimización de Ligas

![Status](https://img.shields.io/badge/Status-En_Desarrollo-green)
![Category](https://img.shields.io/badge/Category-EdTech-orange)

Una plataforma web dedicada a recopilar estrategias, trucos y métodos de optimización para usuarios de Duolingo que desean dominar las ligas competitivas sin sacrificar su tiempo personal.

---

## 🚀 Sobre el Proyecto

Este proyecto nace de la necesidad de equilibrar el aprendizaje de idiomas con la gamificación de la plataforma. Muchos usuarios se sienten frustrados por la alta competitividad de la **Liga Diamante**. Aquí documentamos cómo el algoritmo de emparejamiento puede trabajar a tu favor.

## ✨ Características Principales

* **Calculadora de XP:** Herramienta para planificar cuánto XP necesitas para mantenerte en tu liga actual.
* **Guía de Horarios:** Cronograma para maximizar el uso de los cofres de *Madrugador* y *Noctámbulo*.
* **Estrategia de "Late Join":** Tutorial sobre cómo entrar a las ligas en el momento justo para evitar grupos hiper-competitivos.
* **Blog de Actualizaciones:** Análisis de los cambios constantes en el algoritmo de Duolingo.

---

## 📋 Estrategia Maestra (Wiki)

Actualmente, el pilar central de nuestra web se basa en la siguiente estrategia verificada:

### 1. El "Late Join" (Entrada Tardía)
Para evitar a los usuarios que generan +10,000 XP semanales:
- **Táctica:** No hagas ninguna lección el lunes.
- **Uso de Protectores:** Utiliza hasta 2 protectores de racha para iniciar la liga el **martes o miércoles**.
- **Resultado:** Serás agrupado con usuarios menos intensos, facilitando el ascenso o permanencia.

### 2. Control de Perfil de Usuario
El sistema te empareja según tu historial reciente:
- **Límite sugerido:** Mantente cerca de los **800 - 1000 XP** semanales si tu objetivo es solo permanencia.
- **Efecto:** El algoritmo te mantendrá en "ligas relajadas". Si subes a 5,000 XP una semana, la siguiente será considerablemente más difícil.

### 3. El Ciclo de Doble XP
Maximiza tu tiempo con bonos multiplicadores:
| Bono | Horario de Obtención | Horario de Uso |
| :--- | :--- | :--- |
| **Cofre Madrugador** | 06:00 AM - 12:00 PM | Después de las 06:00 PM |
| **Cofre Noctámbulo** | 06:00 PM - 12:00 AM | Mañana siguiente (antes de las 12 PM) |
| **Misiones Diarias** | Durante el día | Al completar la 3ra misión |

## 🛠️ Tecnologías Sugeridas para la Web

- **Frontend:** React.js o Next.js para una experiencia de usuario rápida.
- **Estilos:** Tailwind CSS (diseño limpio y minimalista).
- **Despliegue:** Vercel o Netlify.
